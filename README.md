# 📊 India IPO Analysis (2010–2021)

A data analysis project exploring 264 Indian IPOs using Python and Tableau.

---

## 🔍 Project Overview

This project analyzes Indian IPO performance data from 2010 to 2021.
The goal is to find patterns — which IPOs succeeded, which failed, and why.

---

## 📈 Key Findings

- ✅ 54% of IPOs gave positive listing gains
- 📅 Best year: **2020** — avg gain of 47.24%
- 📉 Worst year: **2011** — avg loss of 17.26%
- 🏆 Best IPO: **Birla Pacific** +153.5% on listing day
- ❌ Worst IPO: **VKS Projects** -97.15% on listing day
- 💡 High subscription ≠ High listing gain
